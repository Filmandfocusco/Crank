# App Store Review Notes - Crank

**App Purpose:**
Crank is a professional technical reference and workflow utility designed for film industry professionals, including Cinematographers, Camera Assistants (ACs), and Digital Imaging Technicians (DITs). It provides critical technical data and workflow modules required on high-end professional film sets.

**Key Features for Reviewers:**
1. **Camera Database**: A comprehensive, offline-accessible database of professional cinema cameras (ARRI, Sony, RED, Phantom). Each camera entry provides deep technical specifications, including sensor dimensions, supported recording codecs, and maximum frame rates for specific resolutions.
2. **Project & Unit Coordination**: Users can define "Project" parameters (Job Name, Base FPS, Shutter Angle, ISO). These settings synchronize across all modules to ensure technical consistency for the camera department.
3. **Off-Speed & Exposure Engines**: Precise modules for determining shutter compensation and ND filter requirements when shooting at high frame rates or non-standard angles.
4. **Setup Card Export**: Features a unique "Export Setup Card" function that summarizes complex camera configurations into a shareable format for on-set coordination between departments (DIT, Camera, and Script).

**Testing Instructions:**
1. **Navigation**: Use the top navigation bar to switch between the four main sections: **Project**, **Off Speed**, **Exposure**, and **Cameras**.
2. **Cameras Tab**: Tap "Cameras," then use the search bar to find a model (e.g., "ARRI 35"). Tap the card to expand a detailed technical specification table.
3. **Project Tab**: Enter a "Job Name" and set a "Base FPS." These settings will update the logic in the "Off Speed" and "Exposure" modules.
4. **Export Feature**: In the **Project** tab, click "Export Setup Card" to see the sharing interface (used for distribution to the crew).

**Additional Context:**
* **No Account Required**: The app is designed as a standalone professional utility; it does not require an internet connection or user account for its core features.
* **Guideline 4.3(a) Context**: This app is a robust technical reference system. It includes a proprietary camera database and integrated project management features specifically tailored for professional cinematography workflows, moving well beyond the scope of a single-purpose tool.
